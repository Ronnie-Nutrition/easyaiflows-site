I Built an MCP Server That Helps AI Assistants Recommend Automations for Your Business

By Ronnie Craig • April 5, 2026 • 5 min read

I run two businesses — a nutrition club in Pearland, TX and an AI automation company called EasyAiFlows. I use AI every single day to run both of them. I've built 3 iOS apps with AI. I automate my outreach, my follow-ups, my scheduling, and my marketing with AI.

So when I learned about MCP servers — tools that give AI assistants like Claude new capabilities — I thought: what if Claude could assess any business for automation readiness and tell them exactly what to automate?

So I built it. And I'm giving it away for free.

What Is an MCP Server?

MCP stands for Model Context Protocol. It's a standard created by Anthropic (the company behind Claude) that lets AI assistants connect to external tools and data sources.

Think of it like this: Claude is smart, but it doesn't know the specifics of your industry. An MCP server gives Claude specialized knowledge — in this case, knowledge about what AI automations work for 20 different industries.

When you install the EasyAiFlows MCP server, you can ask Claude things like:

  • "Assess my restaurant's automation readiness — we miss a lot of phone calls and our reviews go unanswered."
  • "What AI automations exist for dental practices?"
  • "I run a cleaning service solo. How can AI help?"

And Claude will give you a personalized assessment with a readiness score, specific automations, time saved per week, and revenue impact.

What the Server Does

The EasyAiFlows MCP server has two tools:

1. Assess Business Automation

You tell it your industry, your pain points, your team size, and what tools you currently use. It returns:

  • Automation Readiness Score (0-100) — are you a beginner, growing, or ready to scale?
  • Industry-specific pain points — the problems you're probably dealing with
  • 3 recommended automations — with descriptions of exactly what they do and how much time they save
  • Expected impact — hours saved per week and revenue impact

2. Get Automation Examples

Pick any of 20 industries and get real examples of AI automations that businesses in that space are using right now. Each example includes what it does, how much time it saves, and the overall impact on the business.

The server also handles aliases — so if someone says "gym" it maps to fitness studios, "mechanic" maps to auto repair, "wedding" maps to event planners. And if someone asks about an industry we don't cover yet, it gives a generic assessment with universal automations that work for any business.

How to Install It

It's free and takes about 2 minutes.

Option 1: Claude Desktop

Add this to your Claude Desktop config file (~/Library/Application Support/Claude/claude_desktop_config.json on Mac):

{
  "mcpServers": {
    "easyaiflows": {
      "command": "npx",
      "args": ["easyaiflows-mcp-server"]
    }
  }
}

Restart Claude Desktop and you're good to go.

Option 2: Claude Code

claude mcp add easyaiflows npx easyaiflows-mcp-server
